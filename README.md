 🛒 E-Commerce Management System

 📌 Project Description

The **E-Commerce Management System** is a Java-based desktop application designed to simulate an online shopping platform.
It allows users to register, login, browse products, add items to the cart, and generate purchase receipts.
The system also provides an **admin panel** to manage products and categories.

🚀 Features

* User Registration and Login System
* Product Listing and Category Management
* Add to Cart Functionality
* Shopping Cart Management
* Receipt Generation after purchase
* Admin Dashboard for product management
* Database connectivity for storing user and product data

 🛠 Technologies Used

 Java (Swing) – User Interface
 JDBC– Database Connectivity
 MySQL – Database
 Object-Oriented Programming (OOP)

⚙️ How to Run the Project

1. Clone the repository
2. Open the project in NetBeans / Eclipse / IntelliJ
3. Configure the **MySQL database connection** in `DBConnection.java`
4. Run the main file:

Main.java

5. The application interface will open.
